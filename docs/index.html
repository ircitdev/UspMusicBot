UspMusicBot - это интеллектуальный помощник для создания профессиональных музыкальных треков за 1 минуту без специальных навыков. Проект представляет собой Telegram-бот с веб-интерфейсом, использующий передовые AI-технологии для генерации музыки.

💡
Референс: Проект вдохновлен успешным ботом @gusli_aibot (SoNata - Создать песню)
✨

Основные возможности

🎤

Идея → Музыка

Опишите идею текстом или голосом, бот создаст текст песни и сгенерирует трек в выбранном стиле

📝

Текст → Музыка

Предоставьте готовый текст песни, выберите стиль и получите профессиональный трек с вокалом

🎵

Плейлисты

Сохраняйте созданные треки, создавайте плейлисты и делитесь ими с друзьями

💳

Монетизация

Система балансов с интеграцией YooKassa и CryptoBot для покупки пакетов генераций

🌐

Web App

Полнофункциональный веб-интерфейс с расширенными возможностями управления

🔐

Админ-панель

Управление пользователями, аналитика, рассылки и мониторинг API

👥

API интеграции

🎵

Suno API

Основной

Генерация музыки через AIMLAPI.com провайдер. Стоимость: ~$0.50-$1.50 за трек

  • ✓
    Поддержка всех стилей музыки
  • ✓
    Время генерации: 45-90 сек
  • ✓
    Качество: CD (44.1 kHz, MP3)
🤖

Claude API

Anthropic

Генерация текстов песен с помощью Claude Sonnet 4. Стоимость: ~$0.01-$0.02 за текст

  • ✓
    Профессиональные тексты
  • ✓
    Поддержка русского языка
  • ✓
    Структура: куплеты, припев
🎤

Whisper API

OpenAI

Транскрибация голосовых сообщений. Стоимость: $0.006 за минуту

  • ✓
    Точность: 95%+
  • ✓
    Автоопределение языка
💳

YooKassa

Платежи РФ

Основная платежная система для приема платежей в рублях

  • ✓
    Карты РФ, СБП
  • ✓
    Комиссия: 2.8% + 10₽
💰

Деплой и инфраструктура

Docker Compose

docker-compose.yml 
version: '3.8'

services:
  bot:
    build: .
    container_name: uspmusicbot
    restart: unless-stopped
    env_file: .env
    volumes:
      - ./data:/app/data
      - ./logs:/app/logs
    depends_on:
      - redis

  redis:
    image: redis:7-alpine
    container_name: uspmusicbot_redis
    restart: unless-stopped
    ports:
      - "6379:6379"

  webapp:
    build:
      context: ./frontend
    container_name: uspmusicbot_webapp
    restart: unless-stopped
    ports:
      - "3000:3000"

Быстрый запуск

bash 
# Клонировать репозиторий
git clone https://github.com/uspmusicbot/UspMusicBot.git
cd UspMusicBot

# Настроить переменные окружения
cp .env.example .env
nano .env

# Запустить через Docker
docker-compose up -d

# Проверить логи
docker-compose logs -f bot

Безопасность

  • 🛡️
    Rate Limiting
    Защита от спама через Redis (5 запросов/минуту на пользователя)
  • 🔐
    Валидация данных
    Pydantic модели для проверки всех входных данных
  • 🔑
    Telegram WebApp Auth
    HMAC валидация данных от Telegram WebApp
  • 📝
    Логирование
    Полное логирование всех действий через Loguru
🔒
Best Practices: Все API ключи хранятся в .env файле, который не коммитится в Git. Используется HTTPS для всех внешних запросов.
